A high-fidelity, visually stunning personal portfolio website designed to showcase product management experience, strategic thinking, and leadership skills. Built with modern web design principles, featuring glassmorphism, dark mode aesthetics, and engaging micro-animations.

## 🎨 Design & Features
This portfolio is crafted to leave a premium first impression, built with:
- **Glassmorphism UI**: Semi-transparent, blurred glass-like cards and navigation.
- **Dynamic Micro-animations**: Glitch effects, floating elements, and smooth scroll fade-ins.
- **Modern Typography**: Utilizing the *Outfit* font family for a clean, tech-forward look.
- **Fully Responsive**: Optimized for seamless viewing across all devices (mobile, tablet, desktop).

## 🛠️ Built With
- **HTML5**: Semantic and accessible structure.
- **CSS3**: Custom vanilla CSS, custom properties (variables), Flexbox/Grid layouts, and CSS animations.
- **Vanilla JavaScript**: For scroll reveal effects, dynamic interactions, and glitch animations.
- **[Phosphor Icons](https://phosphoricons.com/)**: Clean, consistent vector icons.
- **[Google Fonts](https://fonts.google.com/)**: Inter/Outfit typography.

## 💻 Getting Started
To view and modify the portfolio locally, no complex build tools are required.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/pm-portfolio.git
   ```
2. **Navigate to the directory:**
   ```bash
   cd pm-portfolio
   ```
3. **Run locally:**
   Simply open the `index.html` file in your preferred web browser, or use a local development server like VS Code's "Live Server" extension for hot reloading.

## 📝 Customization Guide
To make this portfolio your own:
- **Personal Info**: Update `index.html` with your name, bio, and social links.
- **Projects**: Replace the placeholder project cards in the `<section id="projects">` with your actual PM case studies and achievements.
- **Styling**: Tweak the CSS variables in `css/styles.css` (`:root` selector) to adjust the primary colors, background gradients, or glass effect intensities to match your personal brand.
